What is KAI?
KAI is a smart voice-activated assistant embedded in a virtual mobile device simulator. It's designed to guide users through common tasks like sending money, uploading photos, and using various apps. KAI uses speech recognition to understand commands and provides step-by-step guidance with visual highlights on screen.

Available Commands:

Finance (GCash)
"Send money" / "Transfer" / "Help me send" / "Magpadala" / "Padala"
"Receive money" / "Help me receive" / "QR code" / "Activate QR" / "Tumanggap" / "Receive"

Social Media (Facebook)
"Upload" / "Post a photo" / "Post photo" / "Mag upload" / "How do i upload" / "Share a photo"
"React" / "How to react" / "Like a post" / "Mag react" / "Love react" / "How do i react"
"Share a post" / "How to share" / "Mag share" / "I share" / "Share post" / "How do i share"

Apps
"Facebook" / "Open facebook"
"GCash" / "Open gcash"
"Open google"
"Open youtube"
"Calculator"
"Open clock"
"Wikipedia" (optionally add a search term, e.g., "Wikipedia python")

Utilities
"Time" — tells current time
"Date" — tells current date
"Go home" / "Home screen" — returns to home
"Hello" / "Hey" — greeting
All commands support both English and Tagalog/Filipino variations where applicable.
